# ADR 0001: idioma padrão do projeto

## Status

Aprovado

## Contexto

O produto será operado e documentado para um contexto brasileiro, com stakeholders que esperam documentação e interface em português.

## Decisão

Adotar português brasileiro como idioma padrão para:

- documentação
- interface
- nomes funcionais das telas
- descrições de métricas
- comentários de arquitetura

## Consequências

- menor fricção para usuários internos
- maior consistência entre produto, documentação e operação
- possibilidade de código técnico manter identificadores em inglês quando isso reduzir ambiguidade estrutural
