# Roteiro de demonstração

## Objetivo

Mostrar como um analista pode usar o GeoBusiness para avaliar um segmento em uma cidade.

## Sequência sugerida

1. Abrir o Explorador de Segmentos.
2. Selecionar o segmento `Cafeterias`.
3. Ajustar o período para 24 meses.
4. Observar os hexágonos com mais ativos e mais baixas.
5. Abrir o detalhe de uma área.
6. Explicar:
   - ativos
   - churn
   - taxa de fechamento
   - motivos de baixa
   - relação com segmentos complementares
7. Comparar duas áreas.
8. Encerrar mostrando a trilha de qualidade dos dados.

## Mensagem final

O POC não dá uma recomendação cega. Ele organiza evidências históricas para apoiar uma decisão mais consciente.
