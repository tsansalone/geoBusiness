# Web do GeoBusiness

## Objetivo

Oferecer uma experiência analítica em PT-BR para exploração de segmentos, áreas e qualidade dos dados.

## Características do scaffold atual

- interface em português brasileiro
- mapa hexagonal sintético em SVG
- filtros de segmento e métrica
- painel de detalhe de área
- comparação rápida
- painel de qualidade dos dados

## Execução local

```bash
npm install
npm run dev
```

## Evolução prevista

- consumir a API em vez de depender apenas do conjunto sintético local
- adicionar visão temporal com filtros de período
- incluir exportação e comparações mais profundas entre áreas
